Інформаційний веб-сайт ТП «Лівобережна» 35/10 кВ

Практична робота №1 з дисципліни «Основи веб-програмування».

Опис проєкту
Односторінковий веб-сайт (Landing Page) для енергетичного об'єкта. Проєкт демонструє навички роботи з:
- Семантичною версткою HTML5.
- Адаптивним дизайном через Bootstrap 5.
- Стилізацією контенту за допомогою CSS3.

Структура репозиторію
- "index.html" — основна розмітка сторінки.
- "css/styles.css" — власні стилі проєкту.
- "images/" — папка з графічними матеріалами (фото підстанції, схеми, логотип).
