Веб-система моніторингу КРМ ТП «Лівобережна» 35/10 кВ

Практична робота №2 з дисципліни «Основи веб-програмування»

Опис проєкту

Інтерактивний веб-інтерфейс (Dashboard) для динамічного моніторингу показників компенсаторної установки (КРМ) підстанції. Проєкт поєднує в собі інформаційну складову енергетичного об'єкта та функціонал реального часу для відстеження технічних параметрів мережі.

Ключові особливості:

- динамічний моніторинг: симуляція та відображення показників (реактивна потужність, $cos \phi$, напруга, температура) у реальному часі.

- інтерактивна аналітика: візуалізація даних за допомогою бібліотеки Chart.js (лінійні графіки та діаграми станів).

- система сповіщень: візуальна індикація критичних станів (пульсація, зміна кольорів карток) та звуковий сигнал при виході параметрів за межі норми.

- гнучке керування: настроювання періоду оновлення даних та можливість ручного редагування граничних значень параметрів через модальне вікно.

- персоналізація: повноцінна підтримка темної та світлої тем із збереженням вибору в localStorage.

- робота з даними: збереження історії вимірювань у пам'яті браузера та можливість експорту звіту у формат CSV.


Технологічний стек

- Frontend: HTML5 (семантична верстка), CSS3 (Custom Properties, Flexbox, Grid).

- Framework: Bootstrap 5.3 (адаптивність, компоненти).

- JavaScript: ES6+, Chart.js (візуалізація), LocalStorage API (збереження налаштувань та історії).

Структура репозиторію

- index.html — основна структура дашборду та модальних вікон.

- css/styles.css — стилізація інтерфейсу, логіка темної теми та анімації статусів.

- script/script.js — логіка обробки даних, керування графіками та інтерфейсом.

- images/ — графічні матеріали та логотипи.

- sounds/ — аудіофайли для системи тривожного сповіщення.
